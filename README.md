# tumbleweed

how to run:      
git clone https://github.com/aditikhare33/tumbleweed.git           
cd tumble_my_weed
python3 tumble_my_weed.py   

how to play:       
move around and jump with arrow keys      
avoid tumbleweeds moving on the screen
